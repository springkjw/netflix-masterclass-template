# 미션 평가 루브릭

## 1) 기능 완성도 (40점)

- 메인/검색/상세 페이지 동작
- 카드 클릭 라우팅 동작
- 검색어 기반 결과 필터링

## 2) 코드 구조 (30점)

- 컴포넌트 분리 적절성
- 타입 정의 충실도
- 중복 로직 최소화
- 서비스 레이어(mock/api) 분리

## 3) UI/UX 품질 (20점)

- 반응형 대응
- 일관된 스타일 토큰 사용
- 사용자 피드백(로딩/빈 결과) 처리

## 4) 발표/회고 (10점)

- 설계 의도 설명
- 기술 선택 근거
- 개선 계획

## 감점 기준

- `any` 남발, 콘솔 에러 방치
- 하드코딩 중복(같은 JSX 다수 복붙)
- 모바일 화면 붕괴
- `lint`, `typecheck`, `format:check` 실패
